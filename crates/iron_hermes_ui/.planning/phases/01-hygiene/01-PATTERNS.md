### Dioxus Component Declaration
**Source:** `src/main.rs` lines 12–13, 22–23
**Apply to:** `src/app.rs`, `src/components/hero.rs`
```rust
#[component]
pub fn ComponentName() -> Element {
    rsx! { ... }
}
```
Rules: always `#[component]`, always `PascalCase`, always returns `Element`, no `cx`/`Scope`.

**Source:** `src/main.rs` lines 3–6

```rust
const NAME: Asset = asset!("/assets/filename.ext");
```
Rules: declared at module top, `SCREAMING_SNAKE_CASE`, path starts with `/assets/`.

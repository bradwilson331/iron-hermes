**File 1: `src/components/shell/block.rs`** (port of `shell.jsx` lines 94-113 + `app.jsx` `RenderBlock` lines 278-302, per PATTERNS lines 248-312):

```rust
use dioxus::prelude::*;
use crate::state::Block as BlockData;
use super::command_line::CommandLine;
use super::tool_call::ToolCall;

/// Outer chrome for any block kind. Variant-dispatched body. Hover-action row
/// is rendered unconditionally; CSS `:hover` on `.wh-block` controls visibility
/// per CONTEXT D-15. Approach A for Out-body fidelity per UI-SPEC line 432
/// (plain text inside `.wh-block-body`, CSS `white-space: pre-wrap` preserves
/// newlines).
#[component]
pub fn Block(data: BlockData) -> Element {
    let kind_class = data.kind_class();

    let (author, time, exit_code) = match &data {
        BlockData::Cmd { command } => (None, command.time.clone(), None),
        BlockData::Out { author, time, .. } => (author.clone(), time.clone(), None),
        BlockData::Ai  { author, time, .. } => (author.clone(), time.clone(), None),
        BlockData::Ok  { author, time, .. } => (author.clone(), time.clone(), None),
        BlockData::Err { author, time, exit_code, .. } => (author.clone(), time.clone(), Some(*exit_code)),
        BlockData::Tool { .. } => (None, None, None),
    };
    let is_ok  = matches!(data, BlockData::Ok  { .. });
    let is_err = matches!(data, BlockData::Err { .. });
    let is_cmd = matches!(data, BlockData::Cmd { .. });

    rsx! {
        div { class: "wh-block {kind_class}",
            div { class: "wh-block-head",
                if let Some(author) = author { span { class: "wh-author", "{author}" } }
                if is_ok  { span { style: "color: var(--success); font-size: 10px;", "[OK]" } }
                if is_err {
                    span {
                        style: "color: var(--danger); font-size: 10px;",
                        if let Some(code) = exit_code { "exit {code}" } else { "exit 1" }
                    }
                }
                if let Some(t) = time { span { style: "margin-left: auto; font-size: 11px; color: var(--fg-dim);", "{t}" } }
            }
            // Variant-dispatched body
            match data.clone() {
                BlockData::Cmd { command } => rsx! {
                    CommandLine {
                        tokens: command.tokens,
                        time: command.time,
                        cwd: command.cwd,
                        glyph: command.glyph,
                    }
                },
                BlockData::Tool { call } => rsx! {
                    ToolCall {
                        name: call.name,
                        args_summary: call.args_summary,
                        status: call.status,
                    }
                },
                BlockData::Out { text, .. } => rsx! { div { class: "wh-block-body", "{text}" } },
                BlockData::Ai  { markdown, .. } => rsx! { div { class: "wh-block-body", "{markdown}" } },
                BlockData::Ok  { message, .. } => rsx! { div { class: "wh-block-body", "{message}" } },
                BlockData::Err { message, .. } => rsx! { div { class: "wh-block-body", "{message}" } },
            }
            // Hover-action button row (always rendered; CSS controls visibility per D-15).
            // Cmd shows copy + rerun; non-Cmd shows copy + rerun + share (UI-SPEC line 351).
            div { class: "wh-block-actions",
                button { class: "wh-icon-btn", title: "copy", "⎘" }
                button { class: "wh-icon-btn", title: "rerun", "↻" }
                if !is_cmd {
                    button { class: "wh-icon-btn", title: "share", "↗" }
                }
            }
        }
    }
}
```

Notes: `class: "wh-block {kind_class}"` produces `wh-block is-cmd` etc. (UI-SPEC line 188 confirms `is-out` parity). The `match data.clone()` block returns `Element` per arm; Dioxus's macro-expansion accepts arbitrary `match` blocks producing rsx fragments. Approach A for Out body is followed (UI-SPEC line 432: "Default to A").

use dioxus::prelude::*;
use crate::state::PaletteItem;

/// Open-by-default overlay (per CONTEXT D-19). First slash row gets `is-active`
/// per UI-SPEC line 301. Sectioned into "Slash commands" and "Workflows".
///
/// Port of warp2ironhermes/project/app/shell.jsx lines 214-268 per CONTEXT D-01.
/// useState + key handlers (onChange/onKeyDown) are stripped per CONTEXT D-06.
#[component]
pub fn CommandPalette(items: Vec<PaletteItem>, query: String, open: bool) -> Element {
    if !open {
        return rsx! {};
    }
    let slash_items: Vec<&PaletteItem> = items.iter().filter(|p| p.section == "slash").collect();
    let workflow_items: Vec<&PaletteItem> = items.iter().filter(|p| p.section == "workflow").collect();
    rsx! {
        div { class: "wh-pal-overlay",
            div { class: "wh-pal",
                div { class: "wh-pal-search",
                    span {
                        style: "color: var(--accent-primary); font-weight: 700;",
                        "⌘K"
                    }
                    input {
                        placeholder: "Search commands, files, recent…",
                        value: "{query}",
                    }
                    span { class: "wh-kbd", "esc" }
                }
                div { class: "wh-pal-list",
                    div { class: "wh-pal-section", "Slash commands" }
                    for (i, it) in slash_items.iter().enumerate() {
                        div {
                            key: "{it.cmd}",
                            class: "wh-pal-row",
                            class: if i == 0 { "is-active" },
                            span { class: "wh-pal-glyph", "/" }
                            span { style: "color: var(--fg-strong);", "{it.cmd}" }
                            span { style: "color: var(--fg-dim);", "— {it.label}" }
                            span { class: "wh-pal-hint",
                                span { class: "wh-pal-kbd",
                                    for (j, k) in it.kbd.iter().enumerate() {
                                        span { key: "{j}", class: "wh-kbd", "{k}" }
                                    }
                                }
                            }
                        }
                    }
                    div { class: "wh-pal-section", "Workflows" }
                    for it in workflow_items.iter() {
                        div {
                            key: "{it.cmd}",
                            class: "wh-pal-row",
                            span { class: "wh-pal-glyph", "▸" }
                            span { style: "color: var(--fg-strong);", "{it.label}" }
                            span { style: "color: var(--fg-dim);", "{it.cmd}" }
                        }
                    }
                }
            }
        }
    }
}

**Out-of-loop bug fix:** The closure passed to `iter().filter(...)` borrows `p`, and since `Vec<&PaletteItem>` is built before the rsx block, this is fine. Iterating with `for (i, it) in slash_items.iter().enumerate()` over `Vec<&PaletteItem>` makes `it` a `&&PaletteItem` — the `{it.cmd}` interpolation works because Rust auto-derefs through one level. If the executor encounters a borrow-checker complaint, an alternative is to clone via `.cloned()` after filter; but `.collect::<Vec<&PaletteItem>>()` should work cleanly.

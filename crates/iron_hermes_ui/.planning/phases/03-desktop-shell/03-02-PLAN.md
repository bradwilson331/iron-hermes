**File 1: `src/components/shell/command_line.rs`** (port of `shell.jsx` lines 115-130 per PATTERNS lines 315-346 + UI-SPEC token-class table):

The component renders a single row: optional cwd label + glyph + token spans + optional time. Per shell.jsx the function signature in React is `({ cwd = "~/projects/ironhermes", glyph = "❯", parts, time })`; in Rust we accept `cwd: Option<String>`, `glyph: Option<String>`, `tokens: Vec<Token>`, `time: Option<String>` and apply defaults via `unwrap_or_else`.

**Use a type alias to avoid the `crate::state::CommandLine` data struct vs `crate::components::shell::CommandLine` component name collision:**

```rust
use dioxus::prelude::*;
use crate::state::Token;

/// CommandLine row. Renders cwd + prompt glyph + ordered token spans + optional
/// time. Token kind ("bin"/"arg"/"flag"/"str") drives the per-span class
/// (`wh-cmd-bin`/`wh-cmd-arg`/`wh-cmd-flag`/`wh-cmd-str`); the bin token also
/// gets the `wh-cmd` modifier class for the stronger fg-strong + 700-weight
/// styling per `assets/warp-ih.css`.
///
/// Port of `warp2ironhermes/project/app/shell.jsx` lines 115-130 per CONTEXT D-01.
#[component]
pub fn CommandLine(
    tokens: Vec<Token>,
    time: Option<String>,
    cwd: Option<String>,
    glyph: Option<String>,
) -> Element {
    let cwd_text = cwd.unwrap_or_else(|| "~/projects/ironhermes".to_string());
    let glyph_char = glyph.unwrap_or_else(|| "❯".to_string());
    rsx! {
        div { class: "wh-cmdline",
            span { style: "color: var(--fg-dim); font-size: 11px;", "{cwd_text}" }
            span { class: "wh-prompt-glyph", "{glyph_char}" }
            span { style: "flex: 1;",
                for (i, t) in tokens.iter().enumerate() {
                    span {
                        key: "{i}",
                        class: "wh-cmd-{t.kind_class()}",
                        class: if matches!(t, Token::Bin(_)) { "wh-cmd" },
                        if i > 0 { " " }
                        "{t.text()}"
                    }
                }
            }
            if let Some(t) = time { span { class: "wh-cmd-time", "{t}" } }
        }
    }
}
```

**Notes:**
- The `for (i, t) in tokens.iter().enumerate()` form is idiomatic per RESEARCH Pattern 3.
- Class-string interpolation `"wh-cmd-{t.kind_class()}"` is the Dioxus 0.7 idiom (RESEARCH Pattern 2) — no `format!()` macro needed inside `rsx!`.
- The `if i > 0 { " " }` literal-space-between-tokens matches the prototype's whitespace separation in shell.jsx line 124.
- Bin tokens get TWO classes (`wh-cmd-bin` plus `wh-cmd`) via two `class:` attributes — Dioxus concatenates them.
- The component's own name is `CommandLine`, but it imports `Token` (not `CommandLine`) from state — no alias needed.

```rust
use dioxus::prelude::*;
use crate::state::TokenBudget;
use super::scanner::Scanner;

/// Status bar — bottom of the terminal column. Five `.wh-pill` spans
/// (mode/model/provider/tokens/scanner) separated by `.wh-sep` middots, plus
/// a right-aligned `.wh-hint` line. Pill colors come from inline
/// `style: "color: var(--pill-N);"` per the design-tokens.css rotation table
/// (UI-SPEC lines 158-163).
///
/// Port of `warp2ironhermes/project/app/shell.jsx` lines 30-50 per CONTEXT D-01.
#[component]
pub fn StatusBar(
    mode: String,
    model: String,
    provider: String,
    tokens: TokenBudget,
    scanner_active: bool,
    hint: String,
) -> Element {
    let used_k = tokens.used as f32 / 1000.0;
    let max_k = tokens.max / 1000;
    let pct = ((tokens.used as f32 / tokens.max as f32) * 100.0).round() as u32;
    rsx! {
        div { class: "wh-status",
            span { class: "wh-pill", style: "color: var(--pill-0);", "{mode}" }
            span { class: "wh-sep", "·" }
            span { class: "wh-pill", style: "color: var(--pill-1);", "{model}" }
            span { class: "wh-sep", "·" }
            span { class: "wh-pill", style: "color: var(--pill-2);", "{provider}" }
            span { class: "wh-sep", "·" }
            span { class: "wh-pill", style: "color: var(--pill-3);", "{used_k:.1}K/{max_k}K ({pct}%)" }
            span { class: "wh-sep", "·" }
            Scanner { active: scanner_active }
            span { class: "wh-hint", "{hint}" }
        }
    }
}
```

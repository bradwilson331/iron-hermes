IronHermes Unicode Glyph Reference
===================================

Separators
  ·   U+00B7  MIDDLE DOT            dot separator between status pills
  ─   U+2500  BOX DRAWINGS LIGHT H  horizontal rule under banners (×40)

Scanner (knight-rider, lit → fade)
  █   U+2588  FULL BLOCK            lit cell (distance 0)
  ▓   U+2593  DARK SHADE            trail distance 1
  ▒   U+2592  MEDIUM SHADE          trail distance 2
  ░   U+2591  LIGHT SHADE           empty / trail distance 3+

Box drawing (HTML-only panel frames)
  │   U+2502  LIGHT VERTICAL
  ┌   U+250C  LIGHT DOWN AND RIGHT
  ┐   U+2510  LIGHT DOWN AND LEFT
  └   U+2514  LIGHT UP AND RIGHT
  ┘   U+2518  LIGHT UP AND LEFT

Literal glyphs
  ^C          literal caret + C    interrupt marker
  ~           literal tilde         home-dir shorthand

Do NOT use:
  ✓ ✗ ✔ ✘     — use [OK] / [MISSING] / [FAIL] instead
  ═ ║ ╔ ╗ ╚ ╝ — double-line box drawing reads decorative
  emoji       — reserved for personality presets, never TUI chrome

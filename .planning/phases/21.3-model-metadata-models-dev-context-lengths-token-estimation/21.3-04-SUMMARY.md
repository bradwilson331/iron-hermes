- Created models_cmd.rs with ModelsSubcommand enum (List, Fetch, Info) following cron.rs pattern
- handle_models_command dispatcher wired into Commands enum and main.rs match block
- cmd_list displays colored table of all models (static + cached) with context lengths, tokenizers, and capabilities per UI-SPEC Surface 1

   - `.env("HOME", std::env::var("HOME").unwrap_or_default())`

   - `.env("IRONHERMES_RPC_ADDR", socket_path.to_str().unwrap())`
   - `.env("IRONHERMES_SESSION_ID", "sandbox")` (per D-05)

   - `.stderr(Stdio::piped())`

   - `.spawn()?`

10. Spawn async tasks to drain stdout and stderr concurrently into `Vec<u8>` buffers (per Pitfall 1 from RESEARCH — prevents deadlock when Python writes large stdout while blocking on RPC).
11. Wrap the combined wait (child.wait() + stdout/stderr drain join) in `tokio::time::timeout(Duration::from_secs(self.config.timeout_secs), ...)`.

`RpcServer::new(listener: UnixListener, dispatch: Arc<dyn ToolDispatch>, max_calls: u32) -> Self`

`RpcServer::serve(self) -> anyhow::Result<()>`:
- Call `self.listener.accept().await` to get a single `UnixStream` (one connection per execution per D-09/D-11).
- Wrap in `BufReader`, read lines via `lines()`.
- For each line (newline-delimited JSON-RPC 2.0 per D-10):
  a. Parse as `serde_json::Value`.
  b. Extract `id`, `method` (String), `params` (Value).
  c. Check method is in `ALLOWED_TOOLS`. If not, write JSON-RPC error response with code `-32601` (method not found).
  d. Check `call_count.fetch_add(1, Ordering::SeqCst) < max_calls`. If limit exceeded, write JSON-RPC error with code `-32000` and message `"RPC call limit exceeded (50 calls)"` (per D-13).
  e. Call `self.dispatch.dispatch(method, params).await`.
  f. On Ok(result): write JSON-RPC success response `{"jsonrpc":"2.0","id":id,"result":result}` + `\n`.
  g. On Err(e): write JSON-RPC error response with code `-32603` (internal error) and message from the error.
  h. Flush the writer after each response.
- When the stream ends (Python exits), return Ok(()) (per D-11 — auto-shutdown on process exit).

**2. Implement `crates/ironhermes-exec/src/hermes_tools.py`:**

This is the Python helper module that scripts import. It uses only Python 3 stdlib (socket, json, os). Per D-08, zero pip install required. Per D-09, calls are synchronous.

```python
"""IronHermes tool bridge — provides agent tool access from Python scripts.

Usage:
    from hermes_tools import web_search, read_file, write_file
    result = web_search("python asyncio tutorial")
    content = read_file("/path/to/file.txt")
"""

import json
import os
import socket


class HermesRpcError(Exception):
    """Raised when an RPC call returns an error."""
    pass


class HermesCallLimitError(HermesRpcError):
    """Raised when the RPC call limit is exceeded."""
    pass


_sock = None
_request_id = 0


def _connect():
    global _sock
    if _sock is None:
        addr = os.environ.get("IRONHERMES_RPC_ADDR")
        if not addr:
            raise HermesRpcError("IRONHERMES_RPC_ADDR not set — not running in IronHermes sandbox")
        _sock = socket.socket(socket.AF_UNIX, socket.SOCK_STREAM)
        _sock.connect(addr)
    return _sock


def _call(method, params):
    global _request_id
    _request_id += 1
    s = _connect()
    req = json.dumps({"jsonrpc": "2.0", "id": _request_id, "method": method, "params": params})
    s.sendall((req + "\n").encode("utf-8"))

    # Read response — accumulate until newline (per Pitfall 4 from RESEARCH)
    buf = b""
    while b"\n" not in buf:
        chunk = s.recv(65536)
        if not chunk:
            raise IOError("RPC connection closed unexpectedly")
        buf += chunk

    line = buf.split(b"\n", 1)[0]
    resp = json.loads(line)

    if "error" in resp:
        code = resp["error"].get("code", 0)
        msg = resp["error"].get("message", "Unknown RPC error")
        if code == -32000:
            raise HermesCallLimitError(msg)
        raise HermesRpcError(msg)

    return resp.get("result", "")


# === Tool functions (D-07 allowed subset) ===

def read_file(path):
    """Read a file and return its contents."""
    return _call("read_file", {"path": path})


def write_file(path, content):
    """Write content to a file."""
    return _call("write_file", {"path": path, "content": content})


def patch(path, diff):
    """Apply a patch/diff to a file."""
    return _call("patch", {"path": path, "diff": diff})


def search_files(pattern, path="."):
    """Search for files matching a pattern."""
    return _call("search_files", {"pattern": pattern, "path": path})


def web_search(query):
    """Search the web and return results."""
    return _call("web_search", {"query": query})


def web_read(url):
    """Read content from a URL."""
    return _call("web_read", {"url": url})


def memory(action, **kwargs):
    """Interact with the agent's memory store."""
    params = {"action": action}
    params.update(kwargs)
    return _call("memory", params)
```

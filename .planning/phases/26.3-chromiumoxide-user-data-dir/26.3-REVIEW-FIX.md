### WR-01: user_data_dir not pre-created when CLI is not the entry point

**Files modified:** `crates/ironhermes-tools/src/browser_session.rs`, `crates/ironagent-tools-api/src/browser_session.rs`
**Commit:** 4cf0055
**Applied fix:** Added `std::fs::create_dir_all(&user_data_dir)` call immediately after `builder = builder.user_data_dir(&user_data_dir)`.

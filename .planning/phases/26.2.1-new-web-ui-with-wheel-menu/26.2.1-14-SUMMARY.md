+/* GAP-26.2.1-07-R3 / D-26.2.1-14-C Branch (c): live DOM diagnostic

+ * (0,2,1) failed to engage visually. Escalate specificity with html

- **Found during:** Task 4 authoring (before commit)
- **Issue:** The plan's proposed test logic used `SITE_CSS.lines().any(|line| ...)` with a single-line predicate (`line.contains("no-scanlines") && line.contains("display") && line.contains("none")`).

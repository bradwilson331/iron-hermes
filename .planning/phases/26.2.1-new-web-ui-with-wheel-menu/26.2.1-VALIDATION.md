- **After every plan wave:** Run `cargo test -p iron_hermes_ui --lib` + `cargo clippy -p iron_hermes_ui --all-features -- -D warnings`
- **Before `/gsd-verify-work`:** Full suite must be green AND `cargo check -p iron_hermes_ui --features legacy-shell` (legacy shell still compiles) AND manual UAT script passes

## Test Infrastructure

| Property | Value |
|----------|-------|
| **Framework** | Rust built-in (`cargo test`) — workspace already configured |
| **Config file** | None — per-crate `Cargo.toml` declares `[[test]]` targets |
| **Quick run command** | `cargo test -p ironhermes-trajectory --lib && cargo test -p ironhermes-core config && cargo test -p ironhermes-state` |
| **Full suite command** | `cargo test --workspace` |
| **Estimated runtime** | ~60s quick / ~5–8 min full (Rust workspace typical) |
| **Build verification** | `cargo build --workspace` (must pass before test) |

---

## Sampling Rate

- **After every task commit:** Run quick command for the affected crate (per Test Map below)
- **After every plan wave:** Run `cargo test --workspace`
- **Before `/gsd-verify-work`:** Full suite must be green; `cargo build --workspace` clean; `cargo clippy --workspace -- -D warnings` clean
- **Max feedback latency:** ~60s for quick command per crate

<objective>
Close CR-03 + WR-01 + IN-03: `workspace_root` persistence currently uses `Path::to_str()` (returns None for non-UTF-8), the prompt-line uses `Path::display()` (lossy U+FFFD), and `cmd_sessions --workspace` filter uses `ws.root.display().to_string()` — three encodings for the same value. Centralize encoding in `Workspace::canonical_root_string() -> String` (lossy, matches Path::display) and replace every persistence + filter site simultaneously. INV-25.3-12 locks the contract against future drift.

Output: One helper, 5 call-site replacements, 3 unit tests, 1 parity-guard test.

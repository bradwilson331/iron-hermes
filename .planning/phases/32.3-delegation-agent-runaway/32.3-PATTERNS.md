**New `RegistrationGuard` struct** (proposed — D-01):
```rust
pub struct RegistrationGuard {
    registry: std::sync::Weak<tokio::sync::RwLock<SubagentRegistry>>,
    id: SubagentId,
}

impl Drop for RegistrationGuard {
    fn drop(&mut self) {
        if let Some(arc) = self.registry.upgrade() {
            tokio::task::block_in_place(|| {
                tokio::runtime::Handle::current()
                    .block_on(async { arc.write().await.unregister_internal(&self.id) })
            });
        }
    }
}
```

**`Weak` ref source pattern** — `SubagentRegistryHandle` wraps `Arc<RwLock<SubagentRegistry>>` (line 127); `Arc::downgrade(&arc)` produces the `Weak` for the guard.

**New registry methods** (D-01 + D-03):
```rust
// register_guarded replaces register at all call sites
pub fn register_guarded(
    &mut self,
    info: SubagentInfo,
    weak: std::sync::Weak<tokio::sync::RwLock<SubagentRegistry>>,
) -> RegistrationGuard {
    let id = info.id.clone();
    self.active.insert(id.clone(), info);
    RegistrationGuard { registry: weak, id }
}

// unregister renamed to unregister_internal, visibility tightened
pub(crate) fn unregister_internal(&mut self, id: &str) -> Option<SubagentInfo> {
    self.active.remove(id)
}
```

### block_in_place Async-to-Sync Bridge
**Source:** `crates/ironhermes-agent/src/subagent_registry.rs` lines 166–216 (`SubagentRegistryHandle` impl)
**Apply to:** `RegistrationGuard::drop`, `ShrikeService` methods, all new `SubagentListSnapshot` trait implementations
```rust
tokio::task::block_in_place(|| {
    tokio::runtime::Handle::current().block_on(async { self.0.write().await.<method>(...) })
})
```
**Constraint:** Only safe on tokio multi-thread runtime. Tests that trigger Drop must use `#[tokio::test(flavor = "multi_thread", worker_threads = 2)]`.

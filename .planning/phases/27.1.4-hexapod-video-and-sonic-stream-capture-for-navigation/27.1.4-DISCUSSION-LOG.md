## Claude's Discretion

- **camera_pan/camera_tilt joint-axis problem:** `CMD_CAMERA#x#y\n` requires both axes simultaneously. Planner decides whether to use midpoint defaults for the unused axis or collapse into a single `camera_move(x, y)` action.
- JPEG validation before base64 encoding (JFIF/Exif header check like Python client's `is_valid_image_4_bytes`).
- Whether to add a read timeout for the video frame read (analogous to 3s read timeout in `read_distance`).

## Deferred Ideas

- Background streaming mode (`start_stream` / `stop_stream`): continuous video at configurable frame rate into ring buffer. Bigger scope, not needed for single-frame navigation decisions.
- `CMD_LED_MOD` animated modes: remain blocked per Phase 27.1.1 D-16.

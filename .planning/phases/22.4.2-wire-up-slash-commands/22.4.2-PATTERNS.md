#### 1B. None-guard + async bridge via `block_in_place` (handlers that call async subsystem methods)

**Analog:** `cmd_stop` (`crates/ironhermes-core/src/commands/handlers.rs` lines 105-136)

```rust
fn cmd_stop(ctx: &CommandContext) -> CommandResult {
    let pr = match &ctx.process_registry {
        Some(p) => p.clone(),
        None => {
            if ctx.agent_running.load(Ordering::SeqCst) {
                return CommandResult::Output(
                    "Stopping agent... (note: cancellation token not yet \
                     wired \u{2014} agent may continue)"
                        .to_string(),
                );
            }
            return CommandResult::Output(
                "No agent is currently running. Use Ctrl-C to cancel an \
                 in-flight turn."
                    .to_string(),
            );
        }
    };
    let count_before = pr.tracked();
    tokio::task::block_in_place(|| {
        tokio::runtime::Handle::current().block_on(pr.drain_and_kill());
    });
    CommandResult::Output(format!(
        "Stopped {} background process(es).",
        count_before
    ))
}
```

**Apply to:** `cmd_memory` (async `to_memory_entries()`), `cmd_compress` (async `engine.compress()`).
Pattern: clone Arc first, then `block_in_place(|| Handle::current().block_on(async { ... }))`.

fn handle_mouse_slash(app: &mut App, arg: &str) -> SlashOutcome {
    match arg {
        "on" => {
            // side effect (crossterm execute!)
            app.mouse_capture_enabled.store(true, Ordering::SeqCst);
            SlashOutcome::Handled("Mouse capture: on ...".to_string())
        }
        "off" => {
            // side effect
            app.mouse_capture_enabled.store(false, Ordering::SeqCst);
            SlashOutcome::Handled("Mouse capture: off ...".to_string())

        "" => {
            let state = if app.mouse_capture_enabled.load(Ordering::SeqCst) { "on" } else { "off" };

        }
        other => SlashOutcome::Unknown { input: format!("/mouse {other}"), hint: "Usage: ...".to_string() },
    }
}

## Shared Patterns

### Async Bridge (MANDATORY for all new handlers with async subsystem calls)

**Source:** `handlers.rs` lines 129-131 (cmd_stop) and lines 516-519 (cmd_models_refresh)
**Apply to:** `cmd_memory`, `cmd_compress`, and any Tier D handler that calls async AgentLoop methods.

```rust
tokio::task::block_in_place(|| {
    tokio::runtime::Handle::current().block_on(async {
        // async call here
    })
});
```

**BLOCKING anti-pattern:** Do NOT use `async fn` for handler bodies — `dispatch()` in `handlers.rs` is a sync fn (line 12). Do NOT use `spawn_blocking` for StateStore (sync Mutex, no async I/O — handled inside the existing `block_in_place` from `dispatch_slash_blocking`).

### `is_some()` guard (MANDATORY for every handler reading an Optional ctx field)

**Source:** `cmd_skills` (`handlers.rs` lines 421-426), `cmd_reload_mcp` (`handlers.rs` lines 643-648)
**Apply to:** Every new `cmd_*` function that reads a `ctx.<subsystem>` field.

Pattern:
```rust
let handle = match &ctx.<field> {
    Some(h) => h.clone(),
    None => return CommandResult::Output("<Subsystem> not configured.".to_string()),
};
```

Or for is_some()-only check:
```rust
if ctx.<field>.is_some() {
    // do work
} else {
    CommandResult::Output("<Subsystem> not configured.".to_string())
}
```

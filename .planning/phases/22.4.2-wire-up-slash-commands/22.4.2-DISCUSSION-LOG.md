| Option | Description | Selected |
|--------|-------------|----------|

**Notes:** Single source of truth across gateway + classic-tui + tui_rata. Preview shown for the dispatch_slash post-router hook with 4 helper functions (handle_toggle, handle_app_inspector, handle_session_control, handle_subsystem_mutator).

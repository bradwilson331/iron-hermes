## What Was Built

### Task 1: Real handler bodies (handlers.rs)

Five real handler functions added to `core::commands::handlers`:

| Handler | Description | StateStore API |
|---------|-------------|----------------|
| `cmd_sessions(args, ctx)` | Lists recent sessions; accepts optional `limit` arg (default 20) | `state_store.list_sessions_text(limit)` |
| `cmd_resume(args, ctx)` | Restores a session by name/id; shows listing when no arg | `state_store.get_session_id(name_or_id)` |
| `cmd_save(args, ctx)` | Exports current or specified session | `state_store.export_session_text(session_id)` |
| `cmd_history(args, ctx)` | Shows history from ctx.history snapshot; falls back to StateStore | `ctx.history` + `state_store.history_text(session_id)` |
| `cmd_title(args, ctx)` | Persists session title via StateStore; informational when no store | `state_store.update_title(session_id, title)` |

All five wired into `dispatch()` match table. D-05 guard pattern applied: every handler returns `CommandResult::Output("…not configured.")` when `ctx.state_store` is None.

`todo_stub()` entries removed: `resume`, `history`, `save` (sessions and title were previously separate arms). Test `dispatch_all_todo_stubs_return_not_yet_available` updated accordingly.

### Task 2: invoke_handler collapse (commands.rs + invariants)

`invoke_handler` in `tui_rata/commands.rs` collapsed from 200-line 30-arm match to 11 lines:

```rust
async fn invoke_handler(
    name: &str,
    ctx: &CommandContext,
    router: &CommandRouter,
    args: &[&str],
) -> Result<CommandResult, anyhow::Error> {
    let def = router.commands.iter().find(|c| c.name == name)
        .ok_or_else(|| anyhow::anyhow!("unknown command: {name}"))?;
    Ok(ironhermes_core::commands::handlers::dispatch(def, args, ctx, router))
}
```

`dispatch_slash` updated to extract args via `def.name`-interpolated `strip_prefix` (D-11) and pass them through to `invoke_handler`.

**INV-22.4-31 updated:** Removed old per-name arm assertions; added D-01 delegation assertion (`ironhermes_core::commands::handlers::dispatch(` present), absence assertions for `"agents"/"skills" => CommandResult::Output(`, and absence of `not yet wired in Phase 22.4` fallback.

**INV-22.4-33 inverted in place:** Was "26 per-name arm literals MUST be present + stub_count >= 26". Now: "26 per-name arm literals MUST be ABSENT + stub_count == 0 + fn cmd_sessions/cmd_resume/cmd_save/cmd_history present in core".

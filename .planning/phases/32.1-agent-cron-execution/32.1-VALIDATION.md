## Wave 0 Coverage

The cron-runner crate (`crates/ironhermes-cron-runner/`) is created by **Plan
32.1-05a Task 1**, which is the FIRST task in Wave 3. Plans 01–04 (Waves 1
and 2) operate exclusively on pre-existing crates (`ironhermes-cron`,
`ironhermes-agent`) and require no Wave 0 scaffolding beyond what already
exists.

| Wave 0 Requirement | Satisfied by |
|--------------------|--------------|
| `crates/ironhermes-cron-runner/Cargo.toml` exists | Plan 32.1-05a Task 1 step (a) |
| `crates/ironhermes-cron-runner/src/lib.rs` exists with three module decls | Plan 32.1-05a Task 1 step (c) |

| Updated tests in `crates/ironhermes-cron/src/store.rs` covering `trigger_job` / dynamic grace / anchoring | Plan 32.1-03 Task 1 + Task 2 |

test-target crate `ironhermes-cron-runner` does NOT exist until Wave 3 begins.
Tasks in plans 01–04 verify against pre-existing crates only and are
themselves Wave 0 for the cron-runner crate.

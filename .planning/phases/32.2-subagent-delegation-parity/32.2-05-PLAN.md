## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| cmd_agents output → user terminal | Output is rendered as text; ASCII tree connectors are static prefix strings — no injection vector |
| subagent_tree_json → future HTTP consumer | JSON values are constructed via serde_json::json! macro — no string interpolation; serializable types only (SubagentTreeNode is Clone + serializable via manual mapping) |

## STRIDE Threat Register

| Threat ID | Category | Severity | Component | Disposition | Mitigation Plan |
|-----------|----------|----------|-----------|-------------|-----------------|
| T-32.2-16 | Information Disclosure | low | task_summary in /agents output and JSON | accept | task_summary was already exposed via `list_summary` (pre-plan behavior). Tree view does not increase the disclosure surface; only changes the visual layout. |

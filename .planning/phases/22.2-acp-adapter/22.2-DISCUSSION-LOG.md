## Tool Rendering

| Option | Description | Selected |
|--------|-------------|----------|
| Structured by tool type (Recommended) | File→path+content, terminal→command+output, web→markdown. Type hints for editor. | ✓ |

**User's choice:** Structured by tool type
**Notes:** None — recommended default accepted.

---

| Option | Description | Selected |
|--------|-------------|----------|

**User's choice:** Strip always
**Notes:** None — recommended default accepted.

---

| Option | Description | Selected |
|--------|-------------|----------|

**User's choice:** Path + full new content
**Notes:** None — recommended default accepted.

```rust
pub(crate) fn wrapped_line_count(line: &str, width: usize) -> usize {
    if line.is_empty() { return 1; }
    let cols = line.chars().count();
    if width == 0 { return 1; }
    (cols + width - 1) / width
}
```

#### D-11 — Paused indicator in `scroll_indicator()`

**Change surface:** `scroll_indicator()` lines 398–405.

**Current code:**
```rust
pub fn scroll_indicator(&self, area: Rect) -> String {
    if self.auto_follow {
        "live".to_string()
    } else {
        let max = self.transcript_max_scroll(area);
        format!("scroll {}/{}", self.transcript_scroll, max)
    }
}
```

**Extended pattern** — add a "paused" branch when `!auto_follow` AND streaming active. The RESEARCH.md recommends Option B (derived from state, no new field): `N = transcript_max_scroll(area) - transcript_scroll`. This avoids a new App field and correctly represents unseen scroll units:
```rust
pub fn scroll_indicator(&self, area: Rect) -> String {
    let max = self.transcript_max_scroll(area);
    if self.auto_follow {
        "live".to_string()
    } else if self.pending_rx.is_some() || self.assistant_buffer.is_some() {
        let n = max.saturating_sub(self.transcript_scroll);
        format!("paused ({n} new lines below)")
    } else {
        format!("scroll {}/{}", self.transcript_scroll, max)
    }
}
```

Fields used: `self.pending_rx` and `self.assistant_buffer` — both already accessible in this method's `&self` context.

## Critical Implementation Notes for Planner

1. **D-07 inversion (MUST-READ):** CONTEXT.md says "skip System messages" but `role_style()` (line 872) returns `Some(Color::DarkGray)` for `Role::System` — no role currently returns `None`. The fix for D-07 is to **include** System messages in `transcript_line_count` using `role_style()`, not skip them. The `let Some(color) = color else { continue }` guard is kept structurally but currently never fires.

3. **`End` key arm placement:** Must be inserted before the `_ => textarea.input(key)` catch-all (line 500) and after the `PageDown` arm (line 494). Match arm order is significant.

5. **`transcript_line_count` is already `pub`** (line 426) — ui.rs can call `app.transcript_line_count(area.width as usize)` directly without any visibility change.

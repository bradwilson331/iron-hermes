## Slash Precheck Wiring (BLOCKER-NEW-03)

Two enforcement points in `app.rs`:

**Primary — `dispatch_or_submit()`** (called from Enter arm in `handle_key`):
```rust
fn dispatch_or_submit(&mut self) {
    let text = self.textarea.lines().join("\n");
    if text.starts_with('/') {
        self.dispatch_slash_blocking(&text);
        self.clear_textarea();
        return;   // never reaches submit()
    }
    self.submit();
}
```

**Defensive re-check — `submit()`** (paranoid redundancy):
```rust
pub fn submit(&mut self) {
    let text = self.textarea.lines().join("\n");
    if text.is_empty() { return; }
    if text.starts_with('/') {
        self.dispatch_slash_blocking(&text);
        self.clear_textarea();
        return;   // never creates pending_rx as User
    }
    // ... push to history, create (tx, rx), set pending_rx/pending_tx
}
```

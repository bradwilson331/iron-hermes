## Viewport & migration shape

### Q1 — Viewport mode

| Option | Description | Selected |
|---|---|---|
| Alternate screen (tmon-style) | EnterAlternateScreen + mouse capture; clean redraws; scrollback lost on /quit | ✓ |
| Inline viewport (ratatui Viewport::Inline) | Renders fixed-height region; transcript flows into terminal scrollback; no full-height chrome | |
| Hybrid — alt screen during turn, dump on exit | Alt-screen session then dump full transcript to stdout on /quit | |

**User's choice:** Alternate screen (tmon-style).

### Q2 — Migration shape

| Option | Description | Selected |
|---|---|---|
| Hard cutover — delete tui/, rewrite run_chat | One big PR, cleanest end state, no split-brain | |
| Side-by-side (tui_rata/) — keep old path compilable for one phase | New module; old tui/ stays alive; deletion in follow-up after UAT | ✓ |
| Feature-flagged at compile time | `--features ratatui-tui` build toggle | |

**User's choice:** Side-by-side (tui_rata/).

### Q3 — Toggle mechanism

| Option | Description | Selected |
|---|---|---|
| Ratatui default; --classic-tui flag restores old path | Commits to ratatui; old path is emergency escape | ✓ |
| Classic is default; --ratatui flag opts into new path | Safer no-behavior-change default | |
| Config-driven via cli-config.yaml | Persistent backend choice with wizard surface | |

**User's choice:** Ratatui default; `--classic-tui` flag + `IRONHERMES_CLASSIC_TUI` env var.

### Q4 — Non-TTY fallback

| Option | Description | Selected |
|---|---|---|
| Auto-fall-back to classic-tui path silently | IsTerminal gate on stdin+stdout | ✓ |
| Fail with a clear error pointing at --classic-tui | Explicit but annoying for pipelines | |
| Fall back to a minimal plain-stdout REPL (no TUI at all) | Third code path for truly headless automation | |

**User's choice:** Auto-fall-back silently.

## Streaming-token wiring

### Q17 — Streaming publish mechanism

| Option | Description | Selected |
|---|---|---|
| tokio::sync::mpsc::UnboundedSender<StreamEvent> | Per-turn channel; main loop selects on recv() | ✓ |

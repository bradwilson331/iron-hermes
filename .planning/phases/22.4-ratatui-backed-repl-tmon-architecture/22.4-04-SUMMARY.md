## What Was Built

### stream_events.rs

`pub enum StreamEvent` with `#[derive(Debug)]` and exactly 8 D-17 canonical variants:

- `Started` — fired once at turn start
- `Delta(String)` — per-token LLM stream delta
- `ToolCall { name: String }` — tool invocation signal
- `ToolProgress { name: String, phase: String }` — long-running tool progress
- `ToolResult { name: String, ok: bool }` — tool completion
- `Finished` — clean turn end
- `Error(String)` — turn error (user-facing string)
- `Cancelled` — user-initiated turn cancellation

Used over `tokio::sync::mpsc::UnboundedSender<StreamEvent>`; channel close signals turn end.

### history.rs

`pub struct ReplHistory` with:

- `load(path, max)` — reads LF-delimited file, decodes `\u{1F}` → `\n`, caps oldest-first
- `save(&mut self, path)` — encodes `\n` → `\u{1F}`, creates parent dirs
- `push(entry)` — dedupe-consecutive + cap enforcement + cursor reset
- `prev()` / `next()` — arrow-key cursor recall (None = back at prompt)
- `reset_cursor()`, `len()`, `is_empty()`, `is_dirty()`
- `DEFAULT_MAX = 1000` (D-07)
- `UNIT_SEP = '\u{1F}'` (D-08)

Backward-compatible with legacy rustyline single-line history files (no `\u{1F}` → loaded unchanged).

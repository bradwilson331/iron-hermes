## Changes Made

### Task 1: repl_input.rs worker wiring

**Files modified:** `crates/ironhermes-cli/src/repl_input.rs`

**Lines modified:**
- Import block (~line 52): added `use rustyline::config::Configurer;`
- Worker init block (~line 244): replaced `let _ = history_path;` drop site with history config + load block
- `Command::Shutdown` arm (~line 297): replaced `break` with save-then-break

**What was added:**
- `use rustyline::config::Configurer` — required by `set_max_history_size` and `set_history_ignore_dups` (trait methods)
- `rl.set_max_history_size(1000)` — cap at 1000 entries
- `rl.set_history_ignore_dups(true)` — ignore consecutive duplicates (corrected API)
- `rl.load_history(path)` on init with explicit `NotFound` silent branch + `tracing::warn!` for other IO errors
- `rl.save_history(path)` in `Command::Shutdown` arm with `tracing::warn!` on error, shutdown proceeds regardless

**Corrected API confirmed:** `set_history_duplicates(HistoryDuplicates::Prev)` does NOT exist in rustyline 15.0.0. The correct method is `set_history_ignore_dups(true)` on the `Configurer` trait.

**NotFound handling confirmed:** `rl.load_history` returns `Err(ReadlineError::Io(NotFound))` on missing file — NOT a silent no-op. The implementation explicitly matches `NotFound` and ignores it so first-run launches produce no warnings.

**Tracing:** `tracing::warn!` used (not `eprintln!`) since the worker runs concurrently with readline and direct stderr writes would corrupt the prompt. `tracing` is a direct dependency in `ironhermes-cli/Cargo.toml` (line 42).

## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 3 - Blocking] Added `use rustyline::config::Configurer` import**

- **Found during:** Task 1 build verification
- **Issue:** `set_max_history_size` and `set_history_ignore_dups` are methods on the `rustyline::config::Configurer` trait. Rust requires the trait to be in scope to call its methods. Without the import, both method calls failed to compile with `error[E0599]: no method named 'set_history_ignore_dups'`.
- **Fix:** Added `use rustyline::config::Configurer;` to the imports block in `repl_input.rs`. This is not a new crate dependency — `rustyline` is already a workspace dep; `config::Configurer` is a re-export within it.
- **Files modified:** `crates/ironhermes-cli/src/repl_input.rs`
- **Commit:** 09a2e7c (folded into Task 1 commit)

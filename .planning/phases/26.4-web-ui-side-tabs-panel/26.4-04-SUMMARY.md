## What Was Built

### `warp_hermes.rs` — Four surgical edits

**Edit 1 — `active_side_tab` signal declaration**
Added `let mut active_side_tab = use_signal(|| 0_usize);` immediately after `active_tab`, before `tokens`. Single global signal per D-09 — not per-session.

**Edit 2 — `on_side_tab_click` closure**
Added `let on_side_tab_click = move |idx: usize| { active_side_tab.set(idx); };` after the `on_tab_*` closure group. Plain `let` (not `let mut`) — the closure itself is not mutated. D-09 invariant: the existing `on_tab_click` body was not touched; it does not reference `active_side_tab`.

**Edit 3 — 4-tuple `config_summary` destructure**
Widened from the Plan 03 3-tuple `(model_name, provider_name, context_length)` to a 4-tuple `(model_name, provider_name, context_length, memory_enabled)`. Loading/error default extended to `("loading...".to_string(), "...".to_string(), 0_u32, false)`. Eliminates the Plan 03 stub (`memory_enabled: false` hardcoded at call site).

**Edit 4 — 9-prop AgentPanel call site**
Replaced old 1-prop `AgentPanel { messages: messages }` with the full 9-prop call matching Plan 03's signature. `tokens` Signal auto-coerces to `ReadSignal<TokenBudget>`; `active_side_tab` Signal auto-coerces to `ReadSignal<usize>`; `session_id` Signal auto-coerces to `ReadSignal<String>`.

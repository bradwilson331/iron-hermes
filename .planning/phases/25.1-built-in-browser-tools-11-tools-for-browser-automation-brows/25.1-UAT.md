result: issue

  OFF (default config) — `browser_console mode:eval expression:"document.title"` returned
  the page title each time without prompting for approval. Expected: approval_needed envelope,
  no execution. The yolo+approval gate is bypassed for browser_console mode:eval —
  T-25.1-02 (arbitrary JS execution mitigation) is not enforced.
severity: major

  Plan 07's unit tests passed (7/7) — most likely (b): the runtime Config with the actual
  autonomous.yolo value is not reaching the tool.

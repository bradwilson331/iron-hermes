## Key Implementation Notes

### url crate decision
The `url` crate is in the workspace Cargo.toml at version 2 but was NOT in `crates/ironhermes-tools/Cargo.toml`. Rather than add a new per-crate dep for a trivial host-extraction use case, a hand-rolled `extract_host()` was implemented: `url.split_once("://")?.1.split('/').next()?.split(':').next()`. This avoids OQ-4's concern about new workspace deps while satisfying the D-15 allowlist contract.

### Drop semantics / handler_task abort
`close(mut self)` calls `browser.close().await` (CDP Browser.close command) then `handler_task.abort()`. When `BrowserSession` is dropped without calling `close()` (e.g., AgentLoop drop), the `JoinHandle` is dropped which does NOT automatically abort the task — the handler pump task will continue until it encounters an error or the CDP channel closes. For AgentLoop drop path, the chromiumoxide `Browser` drop closes the CDP channel, which causes `handler.next().await` to return `None`, ending the loop naturally. This is safe and consistent with RESEARCH.md drop semantics.

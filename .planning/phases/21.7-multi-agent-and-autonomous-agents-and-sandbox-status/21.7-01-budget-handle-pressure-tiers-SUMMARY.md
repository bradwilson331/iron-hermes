# Phase 21.7 Plan 01: BudgetHandle Pressure Tiers Summary

Wave 1 fills the Wave-0 BudgetHandle shell with production-correct SeqCst atomics: `consume()` implements the canonical AI-SPEC §4b `fetch_sub → if prev==0 then fetch_add else Some(prev-1)` pattern (D-10/PROV-10), `pressure()` maps `used*100/max` floor-division to `PressureTier::{None, Caution70, Warning90, Stop100}` (D-15/PROV-09), and a module-free `advisory_text(PressureTier) -> Option<&'static str>` helper returns `CAUTION_ADVISORY` / `WARNING_ADVISORY` verbatim for Plan 05's turn-boundary injection seam (D-17). A new 79-line concurrent-stress test (`budget_concurrency_stress.rs`) proves — with parent + 4 subagent tokio tasks hammering a shared 10k budget — that the sum of successful consumes equals `max_iterations` exactly, and that a concurrent reader's `used()` samples never exceed `max`. All 12 unit tests + 2 stress tests + the existing E-05 grep gate + the Wave-0 transcript lint pass; total runtime well under the 15s Nyquist per-task bound.

`crates/ironhermes-agent/src/budget.rs:69-81`:

```rust
pub fn consume(&self) -> Option<usize> {
    let prev = self.remaining.fetch_sub(1, Ordering::SeqCst);
    if prev == 0 {
        // Compensate: fetch_sub on 0 wraps to usize::MAX; restore to 0.
        self.remaining.fetch_add(1, Ordering::SeqCst);
        None
    } else {
        Some(prev - 1)
    }
}
```

### Task 1-02 — `pressure()` + `advisory_text()` + 7 tier-boundary tests (commit 136c3dc)

`crates/ironhermes-agent/src/budget.rs:86-111` — `pressure()`:

```rust
pub fn pressure(&self) -> PressureTier {
    if self.max == 0 {
        return PressureTier::None;
    }
    let used = self.used();
    let used_pct = (used * 100) / self.max;
    match used_pct {
        p if p >= 100 => PressureTier::Stop100,
        p if p >= 90  => PressureTier::Warning90,
        p if p >= 70  => PressureTier::Caution70,
        _              => PressureTier::None,
    }
}
```

Module-free helper at file level (not a method) for Plan 05 injection-site ergonomics:

```rust
pub fn advisory_text(tier: PressureTier) -> Option<&'static str> {
    match tier {
        PressureTier::Caution70 => Some(CAUTION_ADVISORY),
        PressureTier::Warning90 => Some(WARNING_ADVISORY),
        PressureTier::None | PressureTier::Stop100 => None,
    }
}
```

Seven tier-boundary tests: `pressure_none_below_70` (0..=69 → None), `pressure_caution70_at_exactly_70` (70 → Caution70), `pressure_warning90_at_exactly_90` (90 → Warning90), `pressure_stop100_at_exhaustion` (100 → Stop100 + subsequent None consumes stay Stop100), `pressure_handles_zero_max_without_panic` (max=0 edge → None, no divide-by-zero), `pressure_non_100_max_tiers_correct` (max=50 cadence at 34/35/45/50), `advisory_text_for_each_tier` (all 4 variants mapped — CAUTION_ADVISORY / WARNING_ADVISORY returned verbatim, no substring drift).

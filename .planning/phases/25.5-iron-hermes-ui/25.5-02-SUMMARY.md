## Decisions Made
- Used Dioxus 0.7.1 `#[get]`/`#[post]` macros instead of plan's `#[server]` — AGENTS.md documents 0.7 fullstack uses these annotations, not the 0.6-era `#[server]`
- Return type `Result<T>` (not `Result<T, ServerFnError>`) per Dioxus 0.7.1 convention
- `StateStore::new` wired with `get_hermes_home().join("state.db")` path since it takes `impl AsRef<Path>`, not `&Config`
- `CommandRouter::new(build_registry())` used instead of plan's direct `build_registry()` on router — registry returns `Vec<CommandDef>`, router wraps it

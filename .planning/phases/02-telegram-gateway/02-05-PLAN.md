**1. Create `multimodal.rs`** — Image and document processing per D-05 through D-08:

```rust
use anyhow::{bail, Result};
use base64::Engine as _;
use tracing::{info, warn};
use crate::telegram::{TelegramAdapter, TgMessage, TgPhotoSize, TgDocument};

const MAX_FILE_SIZE: i64 = 20 * 1024 * 1024; // 20MB per D-08

/// Result of processing a TgMessage's attachments.
pub struct ProcessedAttachments {
    /// Text to prepend to user message (extracted document content).
    pub text_prefix: Option<String>,
    /// Base64-encoded image data URI for vision models.
    pub image_data_uri: Option<String>,
}
```

Methods:

`pub async fn process_attachments(adapter: &TelegramAdapter, msg: &TgMessage) -> Result<ProcessedAttachments>`:
1. If `msg.photo` is Some and non-empty:
   - Select largest photo (last element in Vec — Telegram sorts by size ascending).
   - Check file_size against MAX_FILE_SIZE. If over, bail with "File too large (max 20MB)".
   - Call `adapter.get_file(&photo.file_id)` to get file_path.
   - Call `adapter.download_file(&file_path)` to get bytes.
   - Base64-encode: `base64::engine::general_purpose::STANDARD.encode(&bytes)`.
   - Build data URI: `format!("data:image/jpeg;base64,{}", encoded)`.
   - Return `ProcessedAttachments { image_data_uri: Some(uri), text_prefix: None }`.

2. If `msg.document` is Some:
   - Check file_size against MAX_FILE_SIZE. If over, bail.
   - Call `adapter.get_file(&doc.file_id)` + `download_file`.
   - Match on mime_type:
     - `"application/pdf"` -> Extract text via `pdf_extract::extract_text_from_mem(&bytes)?`. Wrap as `format!("[Document: {}]\n{}", filename, text)`.
     - `"text/plain"` | `"text/markdown"` | `"text/csv"` -> `String::from_utf8_lossy(&bytes)`. Wrap as `format!("[Document: {}]\n{}", filename, text)`.
     - Other -> Return error: `format!("Unsupported document type: {}", mime_type)`.
   - Return `ProcessedAttachments { text_prefix: Some(wrapped_text), image_data_uri: None }`.

3. Otherwise: Return `ProcessedAttachments { text_prefix: None, image_data_uri: None }`.

Note: Add `base64 = { workspace = true }` to gateway Cargo.toml if not already present. Check workspace Cargo.toml for base64 — if missing, add `base64 = "0.22"` to workspace deps.

```rust
async fn run_gateway(cli: &Cli, token_override: Option<String>) -> Result<()> {
    let (_, config) = build_client(cli)?;
    let registry = Arc::new(build_registry());

    // Override token if provided via CLI
    let mut config = config;
    if let Some(token) = token_override {
        config.gateway.platforms
            .entry("telegram".to_string())
            .or_default()
            .token = Some(token);
        config.gateway.platforms
            .get_mut("telegram")
            .unwrap()
            .enabled = true;
    }

    info!("Starting IronHermes Telegram Gateway");
    let runner = ironhermes_gateway::GatewayRunner::new(config, registry);
    runner.start().await
}
```

# Phase 27.1.1 — Security

> Retroactive-STRIDE audit. No formal threat model was authored at plan time; the
> register below was built from implementation files by the `gsd-security-auditor` agent.

---

## Trust Boundaries

| Boundary | Description | Data Crossing |
|----------|-------------|---------------|
| LLM → hexapod_tcp::execute() | LLM-supplied JSON action/direction/speed enters the tool executor | Action enum string, direction enum string, speed integer — all validated before use |
| hexapod_tcp → Freenove TCP server | Wire bytes sent to HEXAPOD_IP:5002 | CMD_* strings with `#`-delimited fields; operator-controlled destination |
| Env var → tool | HEXAPOD_IP read from process environment | IP address (operator-controlled, not user/LLM-controlled) |
| ToolRegistry → LLM schema | Tool schemas exposed to the LLM via get_definitions() | Tool names, descriptions, JSON schemas — filtered by toolset_config |

## Non-Blocking Advisories

These findings do not block advancement but should be addressed before autonomous/unattended operation:

| ADV ID | Severity | Finding | Recommendation |
|--------|----------|---------|----------------|
| ADV-01 | Medium | `send_fire_and_forget` (walk/stop/relax) has no connect timeout. If robot is unreachable, an async task blocks for OS TCP timeout (~75s). Sensor commands ARE protected by the 3s timeout. | Add `tokio::time::timeout(Duration::from_secs(5), ...)` around fire-and-forget connects before any autonomous/unattended use. |
| ADV-02 | Low | Command logging at DEBUG level only. No persistent audit trail for hexapod commands. | Promote to INFO or wire through `HookRegistry` JSONL event log for supervised deployments. |
| ADV-03 | Low | The `execute_code` Python sandbox correctly excludes `hexapod_tcp` via RPC sub-registry, but Python code in the sandbox can open raw TCP sockets to port 5002, bypassing the allowlist. | Requires network namespace isolation or seccomp filtering — out of scope for Phase 1. Address in a security hardening phase before LLM-autonomous robot operation. |

## Decisions Made

1. **Decimal K/M formatting** (`/ 1_000`, `/ 1_000_000`) chosen over binary (`/ 1_024`) â€” matches vendor documentation conventions (Anthropic documents "200k", "1M" context windows). Both are valid; decimal is more recognizable to users.

2. **No AppState extension** (R-1 confirmed) â€” `ModelRegistry` is already reachable via `state.resolver.model_registry()`, a sync accessor on `ProviderResolver`. No new fields added.

3. **`default_id` cloned before iteration** â€” the borrow checker requires `id == default_id.as_str()` comparing a `&str` from the registry (which borrows `&self`) to a local. Cloning avoids lifetime entanglement.

## What Was Built

Three server fns in `crates/iron_hermes_ui/src/server/api.rs` were modified. Each now:

1. Clones `global_app_state().shrike` to obtain an `Option<Arc<ShrikeService>>` — `Send + 'static`, suitable for closure capture.
2. Builds the request body JSON before the closure (owned `serde_json::Value`).
3. Calls `tokio::task::spawn_blocking(move || crate::server::state::api_agents_{kill,interrupt,prune}(shrike.as_deref(), body))`.
4. Maps `JoinError` to `ServerFnError::new(format!("spawn_blocking join error: {e}"))` — no panic propagation to the caller.

Diff summary: `crates/iron_hermes_ui/src/server/api.rs` +32/-6. No other files modified. Zero changes to `ironhermes-agent`, `state.rs`, or `agents.rs`.

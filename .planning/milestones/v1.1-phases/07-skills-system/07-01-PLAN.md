Create `crates/ironhermes-core/src/skills.rs` with the following structures and functions:

1. **SkillFrontmatter** struct (Deserialize):
   - `name: String` (required)
   - `description: String` (required)
   - `version: Option<String>`
   - `author: Option<String>`
   - `license: Option<String>`

2. **SkillRecord** struct (Clone, Debug):
   - `name: String`
   - `description: String`
   - `path: PathBuf` (absolute path to SKILL.md)

3. **parse_skill_md(content: &str) -> Option<(SkillFrontmatter, String)>**:
   - Check content starts with `---` (after trim_start)
   - Skip the opening `---\n`
   - Find the FIRST `\n---` in the remaining text to locate closing delimiter
   - Extract YAML block between delimiters, parse with `serde_yaml::from_str::<SkillFrontmatter>`
   - Extract body as everything after the closing `\n---\n` (trimmed)
   - Return None on any parse failure (no panic)

4. **SkillRegistry** struct:
   - `skills: Vec<SkillRecord>` (private)
   - `pub fn load(cwd: &Path) -> Self` — scans three paths in order per D-02:
     (1) `cwd.join(".ironhermes/skills/")`
     (2) `get_hermes_home().join("skills/")`
     (3) Home dir `~/.agents/skills/` (use `dirs::home_dir()` or `std::env::var("HOME")`)
     For each path: if it exists, read_dir to find subdirectories, look for `SKILL.md` in each.
     Use `HashSet<String>` (lowercase names) for dedup — first-path-wins per D-02.
     Log `tracing::debug!` for skipped duplicates.
   - `pub fn catalog_text(&self) -> String` — per D-04: `"- {name}: {description}"` per line, joined with `\n`
   - `pub fn find(&self, name: &str) -> Option<&SkillRecord>` — case-insensitive lookup via `name.to_lowercase()`
   - `pub fn read_content(&self, name: &str) -> Option<String>` — find skill by name, read SKILL.md from disk, parse with parse_skill_md, return body only
   - `pub fn list(&self) -> &[SkillRecord]` — return slice of all skills
